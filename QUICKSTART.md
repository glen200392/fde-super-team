## Tips for Success

### ‚úÖ DO: Provide Context
```
Good: "Deployment takes 4 hours. Team of 5, deploying 3x/day, 
using GitHub + Jenkins. Bottleneck appears to be in testing."

Bad: "Deployments are slow"
```

### ‚úÖ DO: Specify Constraints
```
"Budget max: $50K"
"Must complete in 6 weeks"
"Cannot change database"
"Need SOC 2 compliance"
```

### ‚úÖ DO: Request Specific Formats
```
"Provide as Python script"
"Create Excel-compatible CSV"
"Generate Gantt chart"
"Show as JSON"
```

### ‚ùå DON'T: Ask Multiple Unrelated Questions
```
Bad: "Optimize deployment AND reduce cloud costs AND fix performance"

Good: Create separate threads for each topic
```

### ‚ùå DON'T: Withhold Critical Information
```

Good: "Fix API performance. Stack: Python/FastAPI. 
Scale: 500 QPS. Latency: 2.5s p95. Target: <500ms. Budget: $20K"
```
